Owner operation manual for ZBrowser.

How to open ZBrowser;
    Go into your terminal and make sure you are in the ZBrowser directory/folder. Once you are in the correct directory, type `python3 Browser.py`.

How to use Zbrowser
    Once you insert your name and type `y` or `Y`, you can search in the search query. `helpme`, `help` will open a help section in the "browser".

How customisable is Zbrowser?
    In ZBrowser it is 100% python you can add search queries. All you need is
    `if Search == "YourSearchQuery":`
        `print("YourTextHere")`
        `continue`
